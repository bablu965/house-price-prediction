# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts house prices based on:
- Area (sqft)
- Number of Rooms
- Location Score

It uses **Linear Regression** from Scikit-learn.

---

## ⚙️ Technologies Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

---

## 📊 Features
- Data visualization using heatmap
- Model training using Linear Regression
- Model evaluation using MSE and R² Score
- User input prediction system

---

## 🧠 Machine Learning Model
- Algorithm: Linear Regression
- Train-Test Split: 80% training, 20% testing

---

## 📈 Output
- Heatmap showing feature correlation
- Scatter plot for Actual vs Predicted prices
- Predicted house price based on user input
